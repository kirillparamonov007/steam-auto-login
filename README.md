# Steam Auto Login

🎮 GUI панель для автоматического входа в Steam аккаунты и запуска игры (например, Bongo Cat) поочередно на нескольких аккаунтах.

## 🎯 Возможности

- ✅ GUI на tkinter с выбором аккаунтов (чекбоксы, выбрать все/снять все)
- ✅ Последовательный вход в выбранные аккаунты
- ✅ Автоматический запуск игры через App ID
- ✅ Настраиваемая задержка перед выходом
- ✅ Убийство всех процессов Steam после завершения
- ✅ Steam Guard коды из `.maFile` (TOTP-генерация)
- ✅ Логирование всех действий
- ✅ Настраиваемые пути и параметры

## 📋 Требования

- **Python 3.8+**
- **Windows 10/11**
- `.maFile` для каждого аккаунта (экспортировать из Steam Guard)

## 📦 Установка

### 1. Скачать проект

```

### 2. Установить зависимости

```

### 4. Запустить программу

```bash

```

## 🚀 Использование

### Первый запуск

1. Запустите `python main.py`
2. При первом старте создастся `accounts.json` с пустым списком
3. Нажмите **Добавить** и заполните поля:
   - **Название** — отображается в списке (любая метка)
   - **Логин Steam** — ваш Steam-аккаунт
   - **Пароль** — пароль от Steam
   - **Выбери .maFile** — кнопка для выбора файла

4. Нажмите **⚙ Сохранить** для сохранения настроек:
   - **Путь к steam.exe** — обычно находится в `C:\Program Files (x86)\Steam\steam.exe`
   - **App ID игры** — для Bongo Cat: `420980`
   - **Задержка (сек)** — время, которое программа ждёт после запуска игры перед выходом

### Запуск последовательности

1. Выберите аккаунты (чекбоксы)
   - Или нажмите **Выбрать все**
2. Нажмите **▶ Запустить**
3. Программа будет:
   - Логиниться на первый аккаунт
   - Запускать игру
   - Ждать (настраиваемое время)
   - Выходить из Steam
   - Переходить на следующий аккаунт
   - И так для каждого выбранного

### Файл конфига (accounts.json)

```json
{
  "accounts": [
    {
      "label": "Main Account",
      "login": "my_steam_login",
      "password": "my_password",
      "mafile": "main.maFile"
    },
    {
      "label": "Alt Account",
      "login": "alt_steam_login",
      "password": "alt_password",
      "mafile": "alt.maFile"
    }
  ],
  "settings": {
    "steam_exe_path": "C:\\Program Files (x86)\\Steam\\steam.exe",
    "game_app_id": "420980",
    "game_launch_delay": 10,
    "mafiles_dir": "mafiles"
  }
}
```

## 🔐 Безопасность

⚠️ **ВАЖНО:** пароли хранятся в `accounts.json` открытым текстом. Соответственно:

- Не оставляй `accounts.json` на общедоступном ПК
- Не коммить `accounts.json` и `mafiles/` в git
- Используй на своём личном компьютере

`.gitignore` уже настроен на исключение чувствительных файлов.

## 🔍 Как получить .maFile

### Способ 1: Steam Guard (мобильный)

1. Откройте Steam Guard на телефоне
2. Нажмите на меню (три точки) → "Show authenticator code"
3. Используйте SteamDesktopAuthenticator или аналог для экспорта

### Способ 2: NebulaAuth

Если уже используете NebulaAuth, .maFile уже есть в его папке.

## 🛠️ Решение проблем

### "steam.exe не найден"

- Укажите путь вручную в настройках
- Или проверьте, что Steam установлен

### "maFile не найден"

- Убедитесь что файл лежит в папке `mafiles/` рядом с `main.py`
- Проверьте имя файла (должно быть `.maFile`)

### "Неверный Steam Guard код"

- Проверьте что `.maFile` правильный
- Убедитесь что часы на компьютере синхронизированы

### Игра не запускается

- Проверьте App ID (для Bongo Cat: 420980)
- Убедитесь что игра установлена через Steam
